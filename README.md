# Travel Planning Portal

A simple static travel planning portal built with HTML, CSS, and JavaScript.

## Features

- Browse popular destinations
- Fill out a travel plan form
- Save itineraries locally using browser storage
- Delete saved itineraries

## Files

- `index.html` - main page markup
- `styles.css` - visual styling and responsive layout
- `script.js` - client-side interactivity and local storage logic

## Run locally

1. Open `index.html` in your browser.
2. Alternatively, serve the folder using a local HTTP server.
